1.Confidencialidade:

Refere-se à proteção de informações contra acesso não autorizado. Utiliza criptografia e controles de acesso para garantir que dados sensíveis sejam visíveis apenas para pessoas autorizadas.

2.Integridade:

Assegura que as informações não sejam alteradas de forma não autorizada. Utiliza técnicas como somas de verificação e hashes para verificar se os dados foram modificados ou corrompidos.

3.Disponibilidade:

Garante que os dados e serviços estejam acessíveis quando necessários. Envolve a implementação de sistemas de redundância, backups e estratégias de recuperação para minimizar o tempo de inatividade.

4.Autenticidade:

Confirma que a identidade de um usuário ou sistema é verificada. Utiliza métodos como senhas, autenticação multifatorial e certificados digitais para garantir que as partes envolvidas são quem dizem ser.

5.Não repúdio:

Impede que uma parte negue a autoria de uma ação ou transação. Utiliza registros e assinaturas digitais para fornecer provas de que uma ação foi realizada por um usuário específico.
